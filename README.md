## 🚀 Getting Started

### ☑️ Prerequisites

Before getting started with AI-Prompt-Video_Generator, ensure your runtime environment meets the following requirements:

- **Programming Language:** Python
- **Package Manager:** Pip


### ⚙️ Installation

Install AI-Prompt-Video_Generator using one of the following methods:

**Build from source:**

1. Clone the AI-Prompt-Video_Generator repository:
```sh
❯ git clone https://github.com/SushantmaniTripathi/AI-Prompt-Video_Generator
```

2. Navigate to the project directory:
```sh
❯ cd AI-Prompt-Video_Generator
```

3. Install the project dependencies:


**Using `pip`** &nbsp; [<img align="center" src="https://img.shields.io/badge/Pip-3776AB.svg?style={badge_style}&logo=pypi&logoColor=white" />](https://pypi.org/project/pip/)

```sh
❯ pip install -r requirements.txt
```




### 🤖 Usage
Run AI-Prompt-Video_Generator using the following command:
**Using `pip`** &nbsp; [<img align="center" src="https://img.shields.io/badge/Pip-3776AB.svg?style={badge_style}&logo=pypi&logoColor=white" />](https://pypi.org/project/pip/)

```sh

```
